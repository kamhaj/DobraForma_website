== Description ==
GymZone Fitness is a flexible, elegant Fitness and Gym WordPress Theme, which is best suits for fitness institutes, gym club, personal trainers and a yoga center. GymZone is one of the most user-friendly free WordPress themes which can easily handle by any type of users with no coding skills to advanced web developers. This Theme is not only SEO friendly but also packed with Social Media Integration. GymZone has a responsive theme layout, which means it can easily fit in smartphones, tablets, and any screen sizes devices. GymZone loads on each and every web browsers and can be translated into numerous local and international languages. Please check theme demo url: https://themestulip.com/demo/gymzone-fitness-pro/, Product Page: https://themestulip.com/themes/gymzone-fitness-wordpress-theme/ , Documentation URL: https://themestulip.com/documentation/gymzone-fitness-pro/

== Frequently Asked Questions ==

= How to customize theme ? =

Just click on Appearance -> Customize
Now you can customize all section in theme

= How to Upgrade theme to pro ?=
Just click on Appearance -> Customize 
Now find upgrade to pro button, just click on it to upgrade pro version

= How to work on Header Phone and Address =
Just click on Appearance -> Customize -> Header Phone and Address

= How to work on Header social links =
Just click on Appearance -> Customize -> gymzone-fitness Social Link

= How to work on Banner Text? =
Just click on Appearance -> Customize -> gymzone-fitness Home Banner Text

= How to work on Footer Widgets ? =
Just click on Appearance -> Customize -> Widget

= How to work on Footer copyright ? =
Just click on Appearance -> Customize -> gymzone-fitness Footer
